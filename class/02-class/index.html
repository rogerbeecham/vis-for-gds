Session outcomes

By the end of this session you should gain the following knowledge:

  • Be able to describe data according to its main characteristics – dimensions and measurement levels.
  • Appreciate the characteristics and importance of Tidy data for data processing and analysis.

By the end of this session you should gain the following practical skills:

  • Load flat file datasets into R via RStudio.
  • Calculate descriptive summaries over datasets.
  • Apply high-level functions in dplyr for working with Tidy data.
  • Create statistical graphics that expose high-level strutucture in data for cleaning purposes.
